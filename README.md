# 项目简介

## 核心功能
- 用户端功能：用户登录注册、电影详情查看、电影影院搜索、在线购票、座位选择、电影点赞与收藏、订单管理
- 管理端功能：用户管理、电影管理、影院管理、影厅管理、电影排片设置、评论管理、订单管理

## 适用场景
适用于电影院线或在线票务平台，为用户提供电影信息查询、购票选座等一站式服务，同时为管理员提供全面的电影院运营管理工具。系统支持用户获取电影信息、选择观影场次、完成购票流程，并允许管理员对电影、影院、排片等核心资源进行有效管理。

## 技术特点
采用前后端分离架构，前端基于Vue.js生态构建，使用vue-router实现路由管理，vuex进行状态管理，element-ui提供UI组件，axios处理HTTP请求；后端采用Node.js 14和Express框架，数据库使用MySQL 5.7，确保系统具有良好的可扩展性和维护性。
